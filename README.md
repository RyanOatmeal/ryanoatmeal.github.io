# ryanoatmeal.github.io
A place for me to doodle with programming.

# Requirements

## node

I prefer to use `nvm` who's instructions you can find [here](https://github.com/nvm-sh/nvm).

## sqlite

https://sqlite.org/index.html

# Installation

1. `nvm install 22`
2. `npm install`
3. `npm run compile`

# Run Locally

`npm run start`
